std::string CLI::zeroAppender ( std::string  time)
throw (std::logic_error
)

Appends a zero to the end of a time string if it needs it(e.g. 13:00 instead of 13:0)

Parameters
std::string- The string containing the time to be evaluated.
Precondition
: A time parameter is in local scope and we know it's length <5 (this function does NOT correct strings with greater length but will disregard them.)
Postcondition
: If necessary, a single zero is appended to the time. If other errors, no action taken.
Returns
: std::string - The time either with a zero appended or in it's original state when passed in.
